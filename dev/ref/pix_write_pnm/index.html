Tesseract.pix_write_pnm — Function
pix_write_pnm(
    filename::AbstractString,
    pix::Pix
)::Bool

Write an image to a file in the PNM image format. If there is an error false is returned.

Arguments:

TNameDefaultDescription
RfilenameThe name of the file to write to.
RpixThe image to write to the file.

Details:

If the file exists it will be overwritten.
